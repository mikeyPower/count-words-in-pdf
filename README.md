# count-words-in-pdf
A python script to count the number of words in a pdf file

## Prerequisite
    $ sudo apt-get install python3
    $ ./install.sh

## Run
    $ python count_pdf_file.py <filename>
    
## Demo
    $ python count_pdf_file.py irelands_call.pdf
    file found!
    133 words in irelands_call.pdf
